# 3-5 Minute Demo Script

## 0:00 - Problem

"A resume tells us what someone claims to know, but not how well they know it. SkillProof AI validates each required skill through evidence and targeted questions."

## 0:30 - Inputs

Upload a JD and resume as PDF, DOCX, TXT/MD, CSV, or XLSX, or paste the text manually. For a quick recorded demo, use **Run sample end-to-end** inside the Inputs tab demo helper.

Explain that the JD defines required skills and the resume provides claimed experience.

Mention the real workflow: hiring teams receive resumes as PDFs/DOCX and often track candidates in CSV/XLSX exports, so the tool is built for operational intake rather than a hand-picked sample.

## 1:10 - Skill Extraction

Open **Inputs**.

Click **Extract required skills** if using custom inputs. Show the extracted skill matrix:

- required skills
- JD priority
- resume evidence count
- question count

Mention that SkillProof uses a curated taxonomy for common skills and a dynamic fallback for role-specific skills, so a judge can paste a marketing, operations, sales, cloud, or product JD and still get assessment questions and learning paths.

## 1:50 - Skill Conversation

Open **Live Assessment**.

Say:

"The assessment is conversational. It asks one skill question at a time, then generates a follow-up from the candidate's actual answer."

Click **Start live assessment**. Show one main question, type or paste a short answer, and point out that SkillProof gives brief natural feedback before asking the next follow-up.

Say:

"This is not a generic questionnaire. With Gemini or OpenRouter configured, questions use this JD, this resume, recent answers, and the current reply. Without an API key, the local fallback still uses the JD sentence and resume evidence, so the interview changes when the input changes."

Mention that the tabs remain available during the demo, but the strongest story is the live assessment plus personalized learning plan.

Click **Score conversation**.

Point at the **Executive Summary** at the top of the app:

- readiness decision
- proof coverage
- main hiring risk
- next proof action
- ROI snapshot

Say:

"This is designed for busy recruiters and judges: in one screen they can see whether the candidate is ready, what proof exists, what risk matters most, and what business value the workflow creates."

## 2:40 - Gap Analysis

Open **Gap Analysis**.

Show:

- skill score
- level
- evidence status
- gap priority
- scoring breakdown
- why the skill was chosen
- why the gap was detected

Say:

"The score is not a black box. It is broken into resume evidence, answer quality, practical depth, and confidence."

## 3:30 - Claim-To-Proof Ledger

Open **Claim-to-Proof Ledger**.

Say:

"This is the differentiator. The app does not just say the candidate knows Python or React. It traces each claim to resume evidence, assessment proof, gap reason, and a proof task."

Show:

- JD priority
- resume proof
- assessment proof
- audit status
- proof task

## 3:55 - Learning Plan

Open **Personalized Learning Plan**.

Show:

- time estimate
- adjacent skill path
- curated course path
- optional AI-personalized roadmap
- hands-on practice drill
- 5-day sprint plan
- proof-of-learning artifact
- learning-style adaptation

Say:

"The learning plan is practical: target level, course path, weekly sprint, drill, and proof artifact, not just a generic list of links."

## 4:20 - ROI Dashboard

Open **ROI Dashboard**.

Show:

- cost saved
- throughput gain
- time saved
- accuracy improved

Say:

"This connects the assessment to measurable business outcomes without changing the core candidate-focused workflow."

Add:

"This is the ROI layer: fewer manual resume checks, better accuracy because every score has evidence, and higher throughput because the same structured workflow can be repeated across candidates."

## 4:50 - Export

Open **Export**.

Show the final Markdown report and download button.

Optionally open **AI reviewer** and say:

"The core product works without an API key so judges can run it reliably. If a Gemini 3 Pro key is configured, the optional AI reviewer reads the structured scores and reason codes to produce semantic calibration notes without changing the auditable rubric."

Close with:

"SkillProof AI turns resume claims into assessed proficiency, visible gaps, and a realistic learning plan."
